## 🎯 Conventions de Nommage

### 📁 Dossiers
- **kebab-case** : `user-service`, `auth-middleware`
- **camelCase** pour les dossiers React : `components`, `contexts`

### 📄 Fichiers
- **PascalCase** pour les composants React : `HomePage.tsx`, `UserCard.tsx`
- **camelCase** pour les services : `userService.js`, `authController.js`
- **kebab-case** pour les configs : `vite.config.ts`, `tailwind.config.js`

### 🔤 Variables et Fonctions
- **camelCase** : `userName`, `getUserData()`
- **SCREAMING_SNAKE_CASE** pour les constantes : `API_BASE_URL`, `ERROR_MESSAGES`
